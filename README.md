# snake
Linux SDL2 Snake game.

## Demo

![Demo](https://github.com/jw4js/snake/raw/main/demo.gif)

## Controls

* W,A,S,D or arrow keys to move
* P to pause
* U to zoom in
* I to zoom out
* Escape to quit
* F11 to toggle fullscreen

## Compiling

Install the following dependencies:

* SDL2

Run the following:

    ./compile
